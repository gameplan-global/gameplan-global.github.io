]]> Comment on GamePlan Software Update – July 13, 2016 by Crane Harvey https://www.gameplan.global/2016/07/13/gameplan-software-update-july-13-2016/#comment-14 Wed, 31 Aug 2016 21:21:50 +0000 https://blogs.gameplan.global/?p=106#comment-14 In reply to Michael Berman.

Re. interactive Gantt – you can drag the Gantt chart bars now, which is a feature that many users are not aware of. However, I expect you are looking for more than this. Let’s take this discussion offline and discuss the exact functionality that you’d like in the Gantt.

Re. the scheduling algorithm, it’s a proprietary algorithm which we have not publicly documented, however I am happy to explain how the algorithm works. One significant difference to other scheduling products is that your resources (people) are always automatically levelled. This means that a person can never be double-booked on two different tasks at the same time, even across different projects. When there are multiple tasks available for a person to work on, they will be assigned to the highest priority task. (In addition to the values in the drop-down list, priorities can be set using integer values from 1 to 1000). Additionally, the scheduling algorithm always honours all other constraints that the user may enter, such as dependencies on other tasks or meetings, an earliest start date and whether the task can be worked on by a single person or if it needs everyone assigned to the task to work on it at one time. One further rule is that tasks are always split around meetings.

There are further scheduling rules and constraints that we could add to the scheduling algorithm, however we are waiting for customer requests to determine which ones are important.
