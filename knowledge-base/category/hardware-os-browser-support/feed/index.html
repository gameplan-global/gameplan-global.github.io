]]> On Windows 8, GamePlan runs in the following web browsers:

  • Chrome
  • Firefox
  • Internet Explorer 11

On Windows 10, GamePlan runs in the following web browsers:

  • Chrome
  • Firefox
  • Edge
  • Internet Explorer 11

On Mac computers, GamePlan runs in the following web browsers:

  • Safari
  • Chrome

Recommended browsers
On a PC, we recommend:

  • Chrome
  • Edge

On a Mac, we recommend:Safari
On a PC, Chrome and Edge use much less memory than Firefox and Internet Explorer, resulting in better performance.
On a Mac, Safari uses less memory than Chrome, resulting in better performance.

Please note that older versions of Internet Explorer before Internet Explorer 11 are not supported.
