]]> You can import data into GamePlan from Google Sheets.When importing from Google Sheets, GamePlan looks for the field names in the first row of the first worksheet. Only the Task column is mandatory. All other fields are optional.  Fields can be in any column order.

Custom field data can be imported into GamePlan.  First, create the custom fields in GamePlan.  Then, make sure that your column headers in your Google Sheet match the custom field names in GamePlan.

Nested groups can be imported using the | character (vertical bar or pipe character) to separate group names.  To link from tasks in other groups, use the | separator between the group name and the task name, as shown below.

Use a semi-colon separator between people where multiple people are assigned to a task.

In the Google sheet, click the Share button.

On the Share dialog, click Copy link.

In GamePlan, choose Import… from the Plan menu.

Choose Google sheets.

Paste the link into the Public share link field.

Press the Preview button to preview the data before import.

Press the Import button to import the data into GamePlan. The tasks, resources, links and other fields will be imported.

GamePlan uses its proprietary visualization algorithm to automatically lay out the tasks on the Whiteboard in a meaningful way so that the user can see and understand the project workflow.

]]> All GamePlan data can be exported to Microsoft Excel.
From the Reports menu, choose Excel report.

Depending on your browser settings, you may be asked to Open or Save the  Excel file.

All data is output to one Excel workbook.  The workbook contains multiple worksheets which contain different GamePlan reports.  All reports in GamePlan are output to Excel.  The Tasks by Person worksheet contains the most detail.  It is the most suitable for doing further calculations on the schedule, such as cost calculations.

The Excel report always has the same name to facilitate linking to the latest scheduling data from other Excel workbooks or other applications.

Because most software can import data from CSV, exporting the GamePlan data to Excel means that you can import GamePlan data into just about any other software with a CSV file.

]]> Tasks can be imported into GamePlan from Microsoft Excel, Microsoft Project and Google Sheets.

Choose Import… from the Plan menu.

Choose Excel and then browse for the Excel (.xlsx) file to import.

When importing from Excel, GamePlan looks for the field names in the first row of the first worksheet.  Only the Task column is mandatory.  All other fields are optional.  Fields can be in any column order.

Custom field data can be imported into GamePlan.  First, create the custom fields in GamePlan.  Then, make sure that your column headers in Excel match the custom field names in GamePlan.

Nested groups can be imported using the | character (vertical bar or pipe character) to separate group names.  To link from tasks in other groups, use the | separator between the group name and the task name, as shown below.
Use a semi-colon separator between people where multiple people are assigned to a task.

After choosing the Excel file, GamePlan displays a preview of the data in the dialog.  This shows the fields that will be imported into GamePlan.  Group nesting is shown in the preview.

Press the Import button to import the data into GamePlan.

GamePlan uses its proprietary visualization algorithm to automatically lay out the tasks on the Whiteboard in a meaningful way so that the user can see and understand the project workflow.
